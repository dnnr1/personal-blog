## About

This is my personal blog project, focused on **simplicity**, **performance**, and a clean developer experience.  
It was built to share posts and experiment with modern **React** and **Next.js** features.

The blog uses a custom backend API to manage content and data.



---

## Tech Stack

### Frontend
- **Next.js** & **React**
- **TypeScript**
- **Tailwind CSS**
- **TanStack React Query**
- **NextAuth.js**
- **React Hook Form** + **Zod**
- **Radix UI** & **Lucide Icons**
- **Markdown Editor** (@uiw/react-md-editor)
- **Theme support** (next-themes)

### Backend

🔗 Backend repository: https://github.com/dnnr1/personal-blog-api

### Deployment
- **Vercel** (Frontend)
- **Oracle / Docker / Docker compose** (Backend)
